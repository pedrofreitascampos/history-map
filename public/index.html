📍 Google Data Guide

⭐ Saved Places (starred / hearted)

Go to takeout.google.com → select Maps (your places) → export. Import the Saved Places.json GeoJSON file.

📋 Saved Lists (Want to go, custom lists)

Go to takeout.google.com → select Saved → export. You'll get one .csv per list (Starred, Want to go, plus each custom list). Drop them here — coords are pulled from each entry's map URL, falling back to OSM geocoding.

🗺️ My Maps

Open your My Map → ⋮ menu → Export to KML. Import the .kml file. Folder names become categories.

📅 Timeline (place visits)

Heads-up: since late 2024 Google moved Timeline to on-device storage. Takeout's "Location History" folder no longer ships your visits — only an encrypted blob and settings files.

To export today, use the Google Maps app on your phone: your profile picture → Your Timeline → ⋮ → Location & privacy settings → Export Timeline data. You'll get a single large .json with semanticSegments. Place names, addresses, place IDs, and per-visit dates are preserved.

Have a pre-2024 Takeout archive? The monthly files in Semantic Location History/ still work — pick them all.

Each visit becomes a "been" place with the actual visit date.
⭐ Google Maps Saved Places

In Google Takeout select Maps (your places). Inside the zip you'll find Saved Places.json and any named lists (e.g. Want to go.json). You can select multiple files at once — "Want to go" lists are imported as wishlist.

Or drag the .json files onto the drop zone above.
📌 Mapstr

In the app: Profile → ⚙️ Settings → Manage your data → Export. You'll receive a CSV or GeoJSON file by email. Import it here — tags and names are preserved.

📱 Android Apps

Apps like "Been There Done That", "Visited", or "Places Been" export as CSV or KML. Use their export feature and import the file here.
